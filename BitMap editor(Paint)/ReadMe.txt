1.Графический редактор(растровый) с возможностью нарисовать изображение, отредактировать
какое-либо другое и сохранить содержимое холста.
Для корректной работы необходимо:
• Python 3.5 и выше
• Размер холста - 1024x600, поэтому изображение для обработки должно быть соответствующим

2.Автор - Михаил Казаков, студент 1го курса ИМКН, КН-102;

3.Запуск редактора - Paint_pyTask_v2.py
Тесты на редактор - Tests_Paint_pyTask.py

4.При запуске программы можно увидеть кнопки, функционал которых говорит сам
за себя (Космос - спрей из кругов разного размера и цвета, соблюденного в особом тоне).
В меню можно сохранить или открыть файл (BMP, EPS, GIF, JPEG, PNG, PNM).

5.В основе всего лежит класс Application.
Выбранный инструмент подсвечивается цветом (карандаш, спрей, ластик, овал, линия, прямоугольник).
При нажатии на «Случайный цвет» кисть будет цвета этой кнопки (каждое нажатие меняет цвет 
кисти и кнопки соответственно).

При выборе цвета на палитре, цвет кисти будет как у кнопки «Палитра».
Ластик - это квадрат белого цвета и как карандаш, космос, спрей зависит от шкалы размера 4-72.

Геометрические фигуры меняют свой цвет в цвет кисти, для того чтобы нарисовать пустую фигуру
необходимо нажать «Палитра», а потом «Отмена» в этом же окне (карандаш будет также без 
цвета в виде круга); рисование фигурами идет по принципу «зажал – отпустил».

Размер кисти можно изменять клавишами «q» -уменьшать размер и «w» - увеличить размер (раскладка 
должна быть английская!)
